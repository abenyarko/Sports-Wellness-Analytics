# Decision Support Dashboard

The interactive Power BI dashboard translates complex statistical analyses into intuitive visualizations that support evidence-based decision-making for coaches, sports performance professionals, and organizational stakeholders.

Designed as the operational component of the longitudinal decision-support framework, the dashboard enables users to monitor athlete wellness, identify meaningful trends, compare player groups, and drill down into individual athlete performance through an interactive reporting environment.

## Dashboard Features

- 📊 Executive season overview with key wellness performance indicators (KPIs)
- 📈 Wellness driver analysis highlighting relationships between primary wellness indicators and overall wellbeing
- 👤 Individual player drill-through reports for athlete-specific monitoring
- 🎛 Interactive filtering by position, athlete status, month, and day of the week
- 📚 Embedded metadata and KPI definitions to promote transparency and consistent interpretation

The dashboard demonstrates how survey statistics and business intelligence can be integrated into a user-friendly decision-support tool that transforms raw wellness data into actionable insights.

Rather than simply displaying data, the dashboard was designed to support timely, evidence-based decisions by transforming complex statistical analyses into intuitive visualizations for coaches and sports performance staff.
